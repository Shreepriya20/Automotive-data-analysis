# 🚗 Automotive Data Analysis Dashboard

An end-to-end automotive telemetry analytics system built using Python and Streamlit, designed to process and analyze time-series vehicle data.

---

## 🔧 Tech Stack
- Python
- Pandas, NumPy
- Streamlit
- Matplotlib
- Object-Oriented Programming (OOP)

---

## 📊 Features
- Processes 10,000+ time-series vehicle records (speed, RPM, throttle, engine temperature)
- Feature engineering and data preprocessing pipelines
- Driving behavior classification (aggressive, normal, braking)
- Anomaly detection (high speed, sudden acceleration, overheating)
- Cruise control analysis using GRA FOC ACTIVE signals
- Interactive Streamlit dashboard for real-time visualization

---

## 🧠 Key Highlights
- Modular OOP-based architecture for scalable data processing
- Simulates real-world automotive CAN bus-like signal analysis
- Designed for production-style analytics workflows
